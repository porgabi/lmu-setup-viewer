LMU Setup Viewer

LMU Setup Viewer is a lightweight desktop application to view and compare LMU setups.

Download latest version Browse all versions

App Preview

Key Features

  • Sectioned setup views aligned with the in-game layout.
  • Side-by-side comparison mode for two setups.
  • Optional difference highlighting with per-section diff counts.
  • Track-based setup grouping in selectors.
  • Track quick-jump index for long setup lists.
  • Class filtering in setup selectors.
  • Custom class sort order for setup listing.
  • Hotkey navigation for faster workflow.
