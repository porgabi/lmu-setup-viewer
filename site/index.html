LMU Setup Viewer

LMU Setup Viewer is a lightweight desktop application to view and compare LMU setups.

Why It Exists

Comparing setups inside LMU can be slow when you only want to inspect values. LMU Setup Viewer focuses on a quick read-and-compare workflow, so you can open setup files and inspect differences faster.

  • LMU-style UX: visual style and layout are inspired by the in-game setup flow.
  • Built for large libraries: virtualized selectors, quick-jump index, and class filters keep long setup lists manageable.
  • Comparison-first workflow: side-by-side setup views with optional diff highlighting and section-level difference status.

How It Works

  1. Set LMU folder once in options.
  2. Select one or two setups from grouped setup lists.
  3. Review and compare values across setup tabs and sections.
  4. Customize your app usage with custom filters and options.

App Preview

App Capabilities

  • Sectioned setup views aligned with the in-game layout.
  • Side-by-side comparison mode for two setups.
  • Optional difference highlighting with per-section diff counts.
  • Track grouping with quick-jump selector index.
  • Class filtering and customizable class sort order.

FAQ

Which platforms are supported?

Windows 10/11 is the primary target. Linux support theoretically exists, but has yet to be tested in practice.

Where are setups read from?

From your LMU installation under UserData/player/Settings after selecting the game folder.

Is LMU Setup Viewer open source?

Yes. The project source code is available publicly on GitHub.

Do I need an account to use it?

No. You can use the app without creating or signing into any account.

How do updates work?

You can check for updates manually at any time, and optionally enable automatic update checks on app launch.

Does the app require internet access?

Only for optional update checks and visiting external links.
