LMU Setup Viewer

LMU Setup Viewer is a lightweight desktop application to view and compare LMU setups.

Designed for Quick Setup Analysis

Comparing setups inside LMU can be slow when you only want to inspect values. LMU Setup Viewer focuses on a quick read-and-compare workflow, so you can open setup files and inspect differences faster.

  • LMU-style UX: visual style and layout are inspired by the in-game setup UI.
  • Built for large libraries: track-based quick-jump index and class filters keep long setup lists manageable.
  • Optimized for comparison: side-by-side setup views with optional diff highlighting and section-level difference status.
  • Built for speed: open the tool and inspect setups without loading into a session.

Features

Side-by-side comparison

Load two setups and inspect differences in aligned sections.

Difference highlighting

Spot changed settings quickly with optional comparison highlights.

Track quick-jump index

Jump directly to the track you're looking for in long lists.

Class filtering

Filter setup lists by class to narrow down what you see.

Custom class-based ordering

Set your preferred class sort order for setup selectors.

Hotkeys and options

Navigate faster and tailor the app's behavior to your preferences.

How It Works

  1. Set your LMU installation folder once in options.
  2. Select one or two setups from setup lists grouped per track.
  3. Review and compare values across setup tabs and sections.
  4. Customize your app usage with custom filters and options.

FAQ

Where are setups read from?

From your LMU installation under UserData/player/Settings after selecting the game folder.

Is LMU Setup Viewer open source?

Yes. The project source code is available publicly on GitHub.

Do I need an account to use it?

No. You can use the app without creating or signing into any account.

Which platforms are supported?

Windows 10/11 is the primary target. Linux support theoretically exists, but has yet to be tested in practice.

How do updates work?

You can check for updates manually at any time, and optionally enable automatic update checks on app launch.

Where can I send feedback?

Feedback, suggestions, and bug reports can be sent to lmu.setupviewer@gmail.com.

Does the app require internet access?

Only for optional update checks and visiting external links.
